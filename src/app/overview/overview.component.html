Overview

1. Use My-Items menu to view pre-added items

2. Use the add item button or Add Items menu to add new items

3. Use edit button to open a dialog which enables you to edit existing data

4. Parent-Child demo menu contains data passing between parent and child component

5. Select rock, paper or scissor to view its details on child component

6. Use edit to edit the name of the item which is reflected in the parent component
